#  Touch Sensing Home Automation PCB

> A PCB-based home automation system that uses touch sensing to control electrical appliances through an intuitive and user-friendly interface.

---

##  Overview

This project presents the design and implementation of a **Touch Sensing Home Automation System** focused on replacing conventional mechanical switches with touch-sensitive controls. The system demonstrates how touch sensing technology can be integrated into home automation to provide a modern, reliable, and convenient method for controlling household electrical appliances.

The project includes circuit design, PCB layout, routing, and 3D PCB visualization, highlighting the complete hardware design workflow.

---

##  Working Principle

The system operates using a touch-sensitive input that triggers the control circuitry. The touch signal is processed through the electronic control circuit, driving the relay to switch connected electrical appliances ON or OFF.

### Workflow

```text
Touch Input
      │
      ▼
Touch Detection Circuit
      │
      ▼
Signal Processing
      │
      ▼
CD4017 Counter & BC547 Driver
      │
      ▼
Relay Switching
      │
      ▼
Electrical Appliance
```
